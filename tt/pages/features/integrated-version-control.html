Enable Feature 'Show Version Control' in Tools -> Prefs. -> Features

Then activate in View

Additional see the following plug-ins:

  • Padre::Plugin::Git works against 0.97+ see -> PadrePluginGit
  • Padre::Plugin::SVN works against 0.96+
  • Padre::Plugin::HG see -> http://bitbucket.org/code4pay/padre-plugin-hg/wiki/Home
  • Padre::Plugin::SVK unknown status

see plugins for more info on Plug-ins

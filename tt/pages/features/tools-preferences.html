My 2ct.:
  • This page is hard to read, s/o should color the changes.
  • Logical/Priority order (0.85) is better then alpha-order.
  • Most of Autocomplete is language-dependent, most languages don't have "methods in packages"
  • "Default_line_ending", "Default_word_wrap_on_for_each_file" and "External window" are global options, not Perl specific
  • "Use_FTP_passive_mode" is a FTP option and should go below the main FTP option
  • "Language-Perl5" should be "Language" -> "Perl5"
  • What happens if the Padre::Plugin::TT would like to add Autocomplete options for Template::Toolkit files to the pref screen ("Language" -> "TT")? Does it work?

[% DEFAULT title="Padre::Plugin::XS" description="" %] [% WRAPPER lib/xhtml_doc %] [% WRAPPER chrome/styled_panel %] This plugin is intended to extend Padre's support for editing XS and C-using-perlapi. Padre::Plugin::XS v0.12 Padre 1.00

Usage

  • install Padre-Plugin-XS: CPAN
  • Enable in Plug-in Manager: To Enable {{{Padre::Plugin::XS}}} in Padre, go to Tools->Plug-in Manager, select **XS Support** in left area.
  • Check that **editor_calltips** is enabled (true) in Tools->Preferences->Advance...
